Why Git is useful to developers.

One of the most useful features of git, is the ability for individuals to edit code in branches that does not change he master branch. This tool can be very useful for projects with many members wanting to push different changes to code, so the master branch can then choose to use attributes from different branches. Essentially each developer can edit to their branch without overriding the master code. Git enhances how developers can work together by using distributed development, pull requests and a stable community to improve workflow.
